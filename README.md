## Featured Projects

### One Piece Explorer — iOS App
[![Swift](https://img.shields.io/badge/Swift-FA7343?style=flat-square&logo=swift&logoColor=white)](https://github.com/Galactic-Avenger/OnePiece)
[![UIKit](https://img.shields.io/badge/UIKit-2396F3?style=flat-square&logo=apple&logoColor=white)](https://github.com/Galactic-Avenger/OnePiece)

A full-featured iOS app built with UIKit and Swift. Pulls live data from a GraphQL API via URLSession to browse characters, crews, and Devil Fruits. Features a custom card-expand navigation transition, a trivia quiz, a wanted poster leaderboard with a hand-built `UICollectionViewLayout`, favorites persistence, and an animated splash screen with audio.

[![Demo](https://img.youtube.com/vi/YBJABlX2ddI/hqdefault.jpg)](https://youtube.com/shorts/YBJABlX2ddI)

### Stats Central — Sports Betting Intelligence App

### NYC Property Sales Analysis (2017–2023)
[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)](https://github.com/Galactic-Avenger/NYC-Property-Sales-Analysis-Project)
[![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)](https://github.com/Galactic-Avenger/NYC-Property-Sales-Analysis-Project)
[![Jupyter](https://img.shields.io/badge/Jupyter-F37626?style=flat-square&logo=jupyter&logoColor=white)](https://github.com/Galactic-Avenger/NYC-Property-Sales-Analysis-Project)
